# Practice for REST user manage using Echo, PostgreSQL

Directories
* auth - Login

* settings - Settings for db, jwt method

Files
* pgdata.zip - pgsql data folder, unzip and run ```pg_ctl.exe -D .\\pgdata -l pglog start```
